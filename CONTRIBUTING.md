# Contribuindo para o SortViz

Obrigado por seu interesse em contribuir com o **SortViz**! Siga as diretrizes abaixo para garantir um processo de contribuição suave.

## Como Contribuir

1.  **Relatar Bugs**: Se você encontrar um bug, abra uma issue descrevendo o problema, passos para reproduzir e seu ambiente.
2.  **Sugerir Melhorias**: Ideias para novos algoritmos ou recursos visuais são bem-vindas via issues.
3.  **Enviar Pull Requests**:
    -   Certifique-se de que seu código segue o estilo do projeto.
    -   Adicione testes para novas funcionalidades.
    -   Atualize a documentação se necessário.

## Estilo de Código

-   Use **Type Hints** em todas as funções.
-   Siga o guia de estilo **PEP 8**.
-   Docstrings devem estar no formato **Google**.

## Ambiente de Desenvolvimento

1.  Crie um ambiente virtual: `python -m venv venv`
2.  Ative o ambiente:
    -   Windows: `venv\Scripts\activate`
    -   Linux/Mac: `source venv/bin/activate`
3.  Instale as dependências: `pip install -r requirements.txt`
4.  Rode os testes: `pytest`
